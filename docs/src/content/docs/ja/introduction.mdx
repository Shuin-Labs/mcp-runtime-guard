## 概要

`mcp-runtime-guard` は、AIエージェント（Claude Code, Cursor など）と実際のMCPサーバーの間に透過的に挟まるCLIプロキシです。

YAMLで定義したポリシーに違反するツール呼び出しをブロックし、すべての操作を監査ログに記録します。

```
Claude Code / Cursor
    │ stdio (JSON-RPC 2.0)
    ▼
mcp-runtime-guard  ← ここでポリシーチェック
    │ stdio (JSON-RPC 2.0)
    ▼
実際のMCPサーバー
```

## なぜ必要か

MCPサーバーはファイルシステム、シェル、外部APIへのアクセスを持ちます。AIエージェントが誤って、あるいは悪意ある指示によって次のような操作を行うリスクがあります：

- `.env` や `~/.ssh/id_rsa` などのクレデンシャルファイルの読み取り
- シェルコマンドの実行
- 意図しない外部サービスへのHTTPリクエスト

`mcp-runtime-guard` はこれらをポリシーで防ぎます。
